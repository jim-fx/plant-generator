# What is Plantarium?

Plantarium is a web app that allows the user to procedurally define and generate 3D plant geometry.

# Developing

### Install prerequisites:

- Node.js
- pnpm

### Install dependencies

```bash
$ pnpm i -r
```

### Start the dev server

```bash
$ pnpm dev
```
